To Experience this System we should follow these instructions or general tips..

** The given API (Currency API) returns an CORS policy error, so I installed an Extension in Chrome to Unblock that
** Command to Run this project
    ** npm install or npm i
    ** gulp
    ** gulp watch .. then it will initiate browser-sync and run the project
    or by manually entering localhost:3000
** 

In this Project I used these technologies
    ** Gulp and some of its plugins
    ** Vanila Javascript (ES6)
    ** JsRender JQuery for Templating the Rendering or run the services
    ** Bootstrap for Responsive UI
